# S.O.L.I.D

Learn about the SOLID principles of object oriented design.

## Focus of this lesson
This lesson will focus on writing code that touches upon the various aspects of the SOLID principles.

## What you need
- An ide of your choice, like visual studio code, or visual studio
- .net core 3.1

## How to get started

* Start by cloning this repository into a folder on your computer. If you've never used git before, you can alternatively use the the "Download ZIP" button to the right.
* Although you have this `README.md` file on your computer it's easier to read it on GitHub, so we recommend you keep this page open with the exercise tasks.

### Symbols and notation used in exercises

#### Icons with special meaning

- :pencil2: - A task you should do
- :book: - A section of text to read (no tasks, just information).
- :bulb: - Additional information.
- :exclamation: - Something important.
- :question: - Open-ended question for the reader ("What do you think would happen if...")
- :poop: - Bad practice (don't-do-this)
- :star: - A bonus task (not required)

#### Keyboard keys

Will look like this:

<kbd>CTRL</kbd> + <kbd>ALT</kbd> + <kbd>C</kbd>

#### Diff blocks

Emphasizes how lines of text should change.

```diff
- this text was removed
+ and replaced with this text
```
